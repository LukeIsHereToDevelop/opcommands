## What's this?

OPCommands is an advanced Discord.js Command Handler that supports slash commands and more!

## Requirements

* Discord.js v13+
* Node.js v16.6.0+

## Installation

```bash
npm install --save opcommands
```

Here is the official Documentation of OPCommands: https://opcommands.lukedev.site
